FileProxy¶

class hep_rfm.FileProxy(source, *targets)[source]¶

Bases: object

Object to store the path to a source file, and some other paths pointing to some target locations. The former is used as a reference to update the latter, using the copy_file() function. Also provides a method to obtain the most accessible file.

path(xrdav=False)[source]¶

Get the most accessible path to one of the files in this class.

Parameters:xrdav (bool) – whether the xrootd protocol is available in root.
Returns:path to the file.
Return type:str

Warning

If the path to a file on a remote site matches that of a local file, it will be returned. This allows to use local files while specifying remote paths. However, if a path on a remote site matches a local file, which does not correspond to a proxy of the path referenced by this object, it will result on a fake reference to the file.
