sync_proxies¶

hep_rfm.sync_proxies(proxies, para_proxies=False, **kwargs)[source]¶

Synchronize a given list of proxies. This function allows to fully parallelize all the processes.

Parameters:
  • proxies (collection(FileProxy)) – file proxies to synchronize.
  • para_proxies (int) – number of processes allowed to parallelize the synchronization of all the proxies. By default it is set to 0, so no parallelization is done (0).
  • kwargs (dict) – extra arguments to FileProxy.sync().
